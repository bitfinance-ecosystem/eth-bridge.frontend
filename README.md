# BitFinance ETH Bridge Frontend
Ethereum<>Harmony<>BitFinance two way bridge (trusted version)



## Install instructions

### Requirements 

* nodejs 

### Commands

* Fetch repo 

```
git clone git@github.com:bitfinance-ecosystem/eth-bridge.frontend.git
```

* Install dependencies

```
npm install
```

* Develop

```
npm run dev
```

* Build

```
npm run build
```

* Start

```
cd build 
serve
```
